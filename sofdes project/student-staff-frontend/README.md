# Campus Lost and Found Student Frontend

React, Vite, and Tailwind CSS frontend for the student/staff lost-and-found interface.

Version: v0.2.0

## Screens

- Login
- Student Dashboard
- Navigation shell
- Lost Report Form
- Found Report Form
- Search Page
- Notifications
- Profile Page
- Claim Page
- Lost Report Details

## Run Locally

```bash
npm install
npm run dev
```

For same-network viewing:

```bash
npm run dev:lan
```

## Scope

- Frontend only
- Mock data first
- No backend code
- Ready for backend API wiring
