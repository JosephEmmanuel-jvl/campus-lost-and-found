# Campus Lost and Found Student Frontend v0.2.0

## Included Screens

- Login
- Student Dashboard
- Report Lost Item
- Report Found Item
- Search
- Notifications
- Profile
- Claim
- Lost Report Details

## Included Behaviors

- Student/staff navigation shell
- React Router routes for all requested pages
- Mock lost reports, found items, notifications, profile, and claim data
- Search filters for keyword, category, location, and report type
- Responsive Tailwind CSS layout
- Vercel and Netlify routing configuration

## Scope

- Frontend only
- React, Vite, and Tailwind CSS
- Mock data first
- No backend, database, or server-side code
